# Language Modes

## Supported Modes

- `zh`
  输出中文版本。保留人名、公司名、产品名和必要术语的英文原文。
- `bilingual`
  先输出中文整理版，再输出英文原文，便于学习和对照。
- `source`
  保留原始语言，不做翻译，只做清洗、分段和格式化。

## Translation Rules

- 只翻译原文表达，不引入新信息。
- 技术术语优先采用常见译法；如果译法不稳定，保留英文并在首次出现时给出中文解释。
- 专有名词不要硬译。
- 中文表达应自然、简洁、避免逐句生硬直译。

## Bilingual Ordering

双语输出建议顺序：

1. 元数据
2. 中文核心要点
3. 中文目录
4. 中文完整文稿
5. 分隔线
6. 英文核心要点
7. 英文目录
8. 英文完整文稿

如果视频很长，也可以只保留中文正文，并在末尾附上英文原始全文。
