# Manuscript Style

## Goal

目标是输出一篇可阅读、覆盖全程的完整文稿，而不是字幕清洗稿、节选稿或逐字核验稿。

## Core Writing Rules

- 保持原始讨论顺序和论点推进。
- 默认覆盖整个可用 transcript，不只整理前半段，也不只提取重点段。
- 合并短句、断裂句和碎片化字幕。
- 删除明显语气词，如“呃”“嗯”“you know”“like”。
- 修正标点、断句和基础语法，使段落自然可读。
- 允许将口语改写成书面表达，但必须保留原意。
- 不补充原视频中没有的新事实、背景或评论。
- 默认不做摘要化压缩；如果原始讨论很长，成稿也应保持接近原始的信息密度。
- 只删除明显噪音：语气词、卡顿重复、自动字幕渐进重复、sponsor/read 与非正文导流内容。

## Heading Rules

- 使用 `H1/H2/H3`。
- 标题应概括主题，而不是复述时间点。
- 标题要可读、具体、可导航。
- 避免“开场”“继续讨论”等低信息量标题。

## Speaker Handling

- 多人对话时，通过自然分段保留问答关系。
- 只有在说话人身份高度可靠时才写姓名。
- 若身份不可靠，宁可不写姓名，也不要硬猜。
- 说话人区分的目标是帮助阅读，不是把文稿改造成逐条对白脚本。

## Forbidden Changes

- 不得颠倒原始顺序。
- 不得将多个观点压缩成一句总结。
- 不得把长段有效讨论删成短版综述。
- 不得为了“像文章”而加入主观评论或补充结论。
- 不得只因为追求简洁，就跳过正常正文段落。
- 不得保留密集字幕式时间戳打断阅读。
