# youtube-transcript

一个根据 YouTube URL 提取视频 transcript，并清理、整理成格式化文稿的 skill。

## 功能

- 根据 YouTube URL 提取 transcript
- 清理 transcript 并整理成格式化文稿
- 支持中文、英文、双语输出
- 支持轻量化 Markdown 和 Word 输出
